## 專題作品說明:

詳細的遊戲分類，讓玩家可以依照自身喜好選擇喜歡的遊戲。

即時的遊戲排行，讓遊戲公司或是代理商可以即時得知自身產品銷售狀況。

多樣的購買來源，讓買家可以有新品或是二手的選擇。

安心的品質把關，杜絕不良二手品，締造買賣家雙贏局面。

快速的問題回饋，以圖文選單並搭配即時的文字客服，讓問題能及時解決。

## 開發使用的技術:

前端: javaScript, jQuery, Ajax, HTML5, CSS3, Bootstrap

伺服器:  Apache Tomcat

資料庫: MySQL, Redis
